# StudyStyle
StudyStyle adalah aplikasi Android untuk membantu mahasiswa mengenali gaya belajar Visual, Auditori, dan Kinestetik (VAK) melalui tes interaktif, dilengkapi fitur login, riwayat hasil, grafik persentase, dark mode, kutipan motivasi, serta rekomendasi bacaan dan metode belajar yang sesuai dengan hasil pengguna.
